## Functional Bug (Critical)

### DETC-01–04, RMVL-02: Detection Skipped on Incremental Swaps

**File:** `src/detection/engine.ts` line 115
**Function:** `runDetectionIfNeeded()`

```typescript
// BUG: swaps.timestamp is Unix seconds; wallet.last_checked_at is Unix milliseconds
gt(swaps.timestamp, lastChecked)  // lastChecked is ~1000x too large — always false
```

**Impact:** After a wallet's initial import, no new scam detection ever fires during monitoring cycles. A wallet that becomes a bundler/dev/sniper after import is never flagged. RMVL-02 (auto-remove on confirmed detection) is also non-functional for post-import behavior.

**Fix:**
```typescript
const lastCheckedSeconds = Math.floor(wallet.last_checked_at / 1000);
gt(swaps.timestamp, lastCheckedSeconds)
```
