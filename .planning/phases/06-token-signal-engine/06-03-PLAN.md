    - 24h eligibility cutoff: twentyFourHoursAgoSec = Math.floor(Date.now() / 1000) - 86400

Implementation steps inside computeAllTokenSignals():

1. Query smart wallets: SELECT address FROM wallets WHERE status='tracked' AND detection_status='confirmed_passing'
   - If none, return { updated: 0, suppressed: 0 }

2. Query distinct token_mints from swaps in last 24h from smart wallets:
   `WHERE wallet_address IN smartAddresses AND timestamp >= twentyFourHoursAgoSec`
   Use `groupBy(swaps.token_mint)` or collect distinct via Set

3. Also query all token_mints that have existing signal records (to detect tokens that need inactive marking):
   `SELECT token_mint FROM token_signals WHERE signal_score > 0`

4. For each token in the union of (recent 24h tokens + existing active records):
   a. Load ALL swaps for this token from smart wallets (no time filter — for current holder calc)
   b. Compute net position per wallet: buyAmt > sellAmt → isCurrentHolder=true
   c. Load wallet_metrics.score_total for each wallet in currentHolders
   d. Load wallet_flags WHERE detector='bundler' AND cleared=false AND wallet_address IN currentHolders
   e. Count buysLast1h and sellsLast1h using oneHourAgoSec cutoff (swaps.timestamp >= oneHourAgoSec AND side='buy'/'sell')
   f. Count totalSmartBuysLast24h (swaps.timestamp >= twentyFourHoursAgoSec AND side='buy')
   g. Build TokenSignalInputs and call computeSignalScore()
   h. Upsert or skip based on result

5. Upsert pattern (drizzle onConflictDoUpdate on token_mint):
   - If result.signalScore > 0: full upsert, increment updated
   - If result.signalScore === 0 AND token had existing record: upsert with signal_score=0, signal_tier='inactive', updated_at=nowMs, increment suppressed
   - If result.signalScore === 0 AND no existing record: skip

Do NOT delete any token_signals rows. Mark stale records with signal_tier='inactive' per design.

Wrap all database reads in a single function (no per-token transactions needed — reads are non-destructive). Upserts can be issued individually per token.

**Part B: CLI signal command (src/commands/signal.ts)**

Create `src/commands/signal.ts` with a `signal list` subcommand:

```typescript
import { Command } from 'commander';
import Table from 'cli-table3';
import chalk from 'chalk';
import { desc, gt } from 'drizzle-orm';
import { db } from '../db/index.js';
import { token_signals } from '../db/schema.js';

export function createSignalCommand(): Command {
  const signal = new Command('signal').description('View token signals');

  signal
    .command('list')
    .description('List top tokens by signal score')
    .option('--limit <n>', 'Number of tokens to show', '20')
    .action((options: { limit: string }) => {
      const limit = parseInt(options.limit, 10);
      const rows = db.select()
        .from(token_signals)
        .where(gt(token_signals.signal_score, 0))
        .orderBy(desc(token_signals.signal_score))
        .limit(limit)
        .all();

      if (rows.length === 0) {
        console.log('No active token signals. Run the monitor loop first.');
        return;
      }

      const table = new Table({
        head: ['Token', 'Score', 'Tier', 'Wallets', 'Buy Vel (1h)', 'Exit Pressure', 'Updated'],
        colWidths: [46, 8, 12, 10, 14, 15, 22],
      });

      for (const row of rows) {
        const tier = row.signal_tier ?? 'unknown';
        const tierColored =
          tier === 'strong' ? chalk.green(tier) :
          tier === 'moderate' ? chalk.yellow(tier) :
          chalk.red(tier);

        const updatedAt = row.updated_at
          ? new Date(row.updated_at).toLocaleString()
          : 'never';

        table.push([
          row.token_mint,
          String(row.signal_score ?? 0),
          tierColored,
          String(row.smart_wallet_count ?? 0),
          String(row.buy_velocity_1h ?? 0),
          String(row.exit_pressure != null ? row.exit_pressure.toFixed(2) : '0.00'),
          updatedAt,
        ]);
      }

      console.log(table.toString());
    });

  return signal;
}
```
