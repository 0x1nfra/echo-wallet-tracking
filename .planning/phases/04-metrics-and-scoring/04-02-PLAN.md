<feature>
  <name>Metric Calculators — win rate, realized PnL, Sharpe ratio, max drawdown, recency score</name>
  <files>
    src/metrics/win-rate.ts,
    src/metrics/pnl.ts,
    src/metrics/sharpe.ts,
    src/metrics/drawdown.ts,
    src/metrics/recency.ts,
    src/metrics/index.ts,
    src/metrics/__tests__/calculators.test.ts
  </files>
  <behavior>
    All inputs are arrays of plain objects (no DB imports). All outputs are numbers.

    **groupIntoClosedTrades(swaps) → ClosedTrade[]**
    - Groups by token_mint; a closed trade = any token with at least one sell and non-null realized_pnl_sol
    - Sum all realized_pnl_sol values for that token; also sum cost_basis_sol for Sharpe input
    - Input: Array<{ token_mint: string; side: 'buy'|'sell'; realized_pnl_sol: number|null; cost_basis_sol: number|null }>
    - Empty sell list → returns []

    **calculateWinRate(closedTrades) → number (0.0–1.0)**
    - Cases: [] → 0; all positive pnl → 1.0; all negative → 0.0; mixed 3 wins 1 loss → 0.75

    **calculateRealizedPnl(swaps) → number**
    - Sum all realized_pnl_sol where side='sell' and realized_pnl_sol is not null
    - Cases: no sells → 0; mixed wins/losses → net sum

    **calculateSharpeRatio(closedTrades) → number**
    - Returns per-trade returns as realized_pnl_sol / cost_basis_sol; requires cost_basis_sol > 0
    - Cases: fewer than 2 valid trades → 0; all same return → cap at 3.0; high-variance high-mean → lower than consistent trader
    - Apply confidence dampener: multiply by min(1.0, trade_count / 50) to penalize thin history

    **normalizeSharpeLike(sharpe) → number (0–100)**
    - Formula: Math.round(((Math.tanh(sharpe * 0.5) + 1) / 2) * 100)
    - Cases: sharpe=0 → 50; sharpe=1.0 → ~76; sharpe=2.0 → ~96; sharpe=-1.0 → ~24

    **calculateMaxDrawdown(swaps) → number (0.0–1.0)**
    - Build cumulative PnL from sells sorted by timestamp; track peak; drawdown = (peak - current) / peak
    - Cases: no sells → 0; monotonically increasing PnL → 0; [+10, +5, +8, +3] cumulative → drawdown from peak

    **calculateRecencyScore(swaps, nowMs?) → number (0–100)**
    - 180-day hard cutoff (WINDOW_MS = 180 * 24 * 60 * 60 * 1000)
    - Cases: 0 recent → 0; 1 recent → 5; 5 recent → 25; 50+ recent → 100
    - Formula: if count=0 → 0; if count<5 → count*5; else → min(100, 25 + (count-5) * (75/45))
  </behavior>

    **WalletScoreResult interface:**
    ```typescript
    export interface WalletScoreResult {
      total: number;                  // 5-95 clamped
      riskAdjustedReturn: number;     // 0-100 sub-score (40% weight)
      winRate: number;                // 0-100 sub-score (20% weight)
      consistencyRecency: number;     // 0-100 sub-score (20% weight)
      activityHealth: number;         // 0-100 sub-score (20% weight)
    }
    ```

    **ComputedMetrics interface (input):**
    ```typescript
    export interface ComputedMetrics {
      sharpeRatio: number;
      winRateDecimal: number;       // 0.0-1.0
      recencyScore: number;         // 0-100
      maxDrawdown: number;          // 0.0-1.0 (percentage as decimal)
      tradeCount: number;           // total swap rows
      recentTradeCount: number;     // swaps in last 180 days
      distinctTokensTraded: number; // unique token_mints with sells in last 180 days
    }
    ```

    **composeScore(metrics: ComputedMetrics) → WalletScoreResult:**
    - riskAdjustedReturn = normalizeSharpeLike(sharpeRatio) [0-100]
    - winRate = Math.round(metrics.winRateDecimal * 100) [0-100]
    - consistencyRecency = combine recencyScore + drawdown penalty:
      - Base = recencyScore (0-100)
      - Drawdown penalty: subtract Math.round(metrics.maxDrawdown * 50) (50% drawdown → -25 points)
      - Clamp to [0, 100]
    - activityHealth = 0.6 * frequencyScore + 0.4 * diversityScore, clamped to [0, 100]:
      - frequencyScore = Math.min(100, recentTradeCount * 2) (50 trades → 100)
      - diversityScore = Math.min(100, distinctTokensTraded * 5) (20 tokens → 100)
    - raw = riskAdjustedReturn * 0.40 + winRate * 0.20 + consistencyRecency * 0.20 + activityHealth * 0.20
    - total = Math.max(5, Math.min(95, Math.round(raw)))

    **Test cases for bundler-vs-genuine-trader separation (ROADMAP success criterion 4):**
    - Bundler profile: winRateDecimal=0.80, sharpeRatio=0.35, maxDrawdown=0.45, recentTradeCount=30, distinctTokensTraded=5, recencyScore=70 → total score should be < 65
    - Genuine trader: winRateDecimal=0.60, sharpeRatio=1.50, maxDrawdown=0.10, recentTradeCount=25, distinctTokensTraded=12, recencyScore=65 → total score should be > 75
    - Genuine trader scores materially higher (difference >= 10 points)
  </behavior>
