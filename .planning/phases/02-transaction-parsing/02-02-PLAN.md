<feature>
  <name>Swap Parser and FIFO Cost Basis</name>
  <files>src/parsers/swap.ts, tests/unit/parsers.test.ts</files>
  <behavior>
    parseSwaps(txs: HeliusTransaction[], walletAddress: string): SwapRow[]
    - Input: array of HeliusTransaction objects + wallet address string
    - Returns array of SwapRow objects (only SOL↔token swaps for supported DEXes)
    - BUY: tx has nativeInput + tokenOutputs → side='buy', sol_amount=nativeInput.amount/1e9, token_amount=tokenOutputs[0].rawTokenAmount.tokenAmount/10^decimals
    - SELL: tx has tokenInputs + nativeOutput → side='sell', sol_amount=nativeOutput.amount/1e9
    - Skip: unknown programId (not in DEX_PROGRAM_IDS_MAP) → return nothing, no error
    - Skip: token-to-token swap (no nativeInput AND no nativeOutput) → return nothing
    - Skip: tx.type !== 'SWAP' → return nothing
    - events.swap may be object or array — always normalize to array and use [0]
    - DEX identification: iterate instructions[].programId against DEX_PROGRAM_IDS_MAP entries
    - cost_basis_sol and realized_pnl_sol are null on all rows from parseSwaps (set by applyFifo)

    applyFifo(swaps: SwapRow[]): SwapRow[]
    - Input: array of SwapRow objects for a single wallet (may span multiple tokens)
    - Sorts by timestamp ASC before processing
    - BUY: push lot { tokenAmount, pricePerToken: sol_amount/token_amount } to Map<tokenMint, lot[]>; sets cost_basis_sol = sol_amount, realized_pnl_sol = null
    - SELL: consume oldest lots FIFO for matching tokenMint
      - Full match: cost_basis_sol = total consumed SOL, realized_pnl_sol = sol_amount - cost_basis_sol
      - Partial orphan (lots run out mid-sell): cost_basis_sol = null, realized_pnl_sol = null
      - Full orphan (no lots at all): cost_basis_sol = null, realized_pnl_sol = null
    - Use epsilon 1e-9 when checking lot exhaustion to avoid float drift
    - Returns new array (does not mutate input)

    Test cases to verify:
    - Buy Raydium → SwapRow with side='buy', sol_amount=1.0, token_amount=10 (1M tokens / 10^5 decimals)
    - Sell Jupiter → SwapRow with side='sell', sol_amount=1.5, token_amount=10
    - TRANSFER tx → empty array
    - Token-to-token → empty array
    - All 5 DEXes identified correctly (RAYDIUM, JUPITER, PUMP_FUN, ORCA, METEORA)
    - Unknown programId → empty array (NOT a result with dex='unknown')
    - FIFO: buy 10 tokens @ 0.1 SOL/token, sell 10 tokens for 1.5 SOL → realized_pnl_sol = 0.5
    - FIFO: sell with no prior buy → cost_basis_sol=null, realized_pnl_sol=null
  </behavior>
