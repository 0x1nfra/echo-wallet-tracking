<task type="auto" tdd="true">
  <name>Task 1: Implement wallet commands (add, remove, list) + CLI wiring</name>
  <files>
    src/commands/wallet.ts

  </files>
  <behavior>
    - add: inserts row with address + optional label; on UNIQUE constraint error prints "Wallet <address> is already tracked." and exits with code 1
    - add --label: stores label in the label column
    - remove: deletes row by address; if address not found prints "Wallet <address> is not tracked." and exits with code 1
    - list (empty): prints "No wallets tracked yet.\n\nGet started: echo wallet add <address>" (no table)
    - list (populated): renders cli-table3 with head ['ADDRESS', 'LABEL', 'STATUS', 'ADDED'], sorted by added_at descending
    - ADDRESS column: truncated as first 8 + '...' + last 4 chars
    - LABEL column: wallet.label value if set, or chalk.gray('(no label)')
    - STATUS column: chalk.green('tracked') for tracked status
    - ADDED column: added_at formatted as locale date string
    - List excludes removed wallets (WHERE status = 'tracked') — use drizzle eq() filter
  </behavior>
  <action>
    1. Create `src/commands/wallet.ts`:

       Structure: export a single `createWalletCommand()` function that returns a Commander `Command`.

       `wallet add <address>`:
       - Accepts `--label <label>` option
       - Wraps db.insert(wallets).values({ address, label: options.label ?? null }).run() in try/catch
       - On `SqliteError` with message containing 'UNIQUE constraint failed': console.error("Wallet " + address + " is already tracked.") then process.exit(1)
       - On success: console.log("Wallet " + address + " added" + (options.label ? " (" + options.label + ")" : "") + ".")

       `wallet remove <address>`:
       - Runs db.delete(wallets).where(eq(wallets.address, address)).run()
       - Check rowsAffected (better-sqlite3 returns `{ changes }` from .run()) — if changes === 0: console.error("Wallet " + address + " is not tracked.") then process.exit(1)
       - On success: console.log("Wallet " + address + " removed.")

       `wallet list`:
       - Query: db.select().from(wallets).where(eq(wallets.status, 'tracked')).orderBy(desc(wallets.added_at)).all()
       - Empty result: console.log("No wallets tracked yet.\n\nGet started: echo wallet add <address>") and return
       - Non-empty: create cli-table3 Table with head ['ADDRESS', 'LABEL', 'STATUS', 'ADDED'] and style { head: ['cyan'] }
       - For each row, push [truncateAddress(row.address), row.label ?? chalk.gray('(no label)'), chalk.green('tracked'), new Date(row.added_at).toLocaleDateString()]
       - console.log(table.toString())

       Helper at top of file:
       ```typescript
       function truncateAddress(addr: string): string {
         return `${addr.slice(0, 8)}...${addr.slice(-4)}`;
       }
       ```

       Import `desc`, `eq` from `drizzle-orm`. Import `Table` from `cli-table3`. Import `chalk` from `chalk`. All relative imports must use `.js` extension (ESM requirement).

    2. Update `src/cli.ts`:
       - Add import: `import { createWalletCommand } from '@/commands/wallet.js';`
       - Add before `program.parse()`: `program.addCommand(createWalletCommand());`
       - Do NOT modify or remove the existing `score` command.

    3. Update `.env.example`:
       - Add the following entries (append, do not overwrite existing content):
         ```
         # Database
         DATABASE_URL=./data/echo.db
         INACTIVITY_DAYS=30
         ```
  </action>
